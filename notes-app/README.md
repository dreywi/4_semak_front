# Notes App

Приложение для создания и управления заметками. Выполнено в рамках дисциплины "Фронтенд и бэкенд разработка" (4 семестр).

## Выполненные работы

| Практическая | Что было сделано |
|--------------|------------------|
| **№13** | • Создание React-приложения с помощью Create React App<br>• Настройка структуры проекта (папки components, pages, utils)<br>• Создание базовых компонентов: NoteCard, NoteForm, NoteList<br>• Настройка маршрутизации с React Router |
| **№14** | • Реализация состояния через useState<br>• Добавление функций создания, редактирования и удаления заметок<br>• Обработка событий (onSubmit, onClick, onChange)<br>• Динамическое обновление списка заметок |
| **№15** | • Интеграция localStorage для сохранения заметок<br>• Загрузка сохранённых данных при запуске приложения<br>• Автоматическое сохранение при каждом изменении<br>• Реализация кастомного хука useLocalStorage |
| **№16** | • Настройка маршрутизации с React Router DOM<br>• Создание страниц: список заметок (главная), просмотр заметки, добавление/редактирование<br>• Использование useParams для получения ID заметки<br>• Навигация между страницами с useNavigate |
| **№17** | • Оптимизация с useCallback и useMemo<br>• Вынос логики в отдельные хуки<br>• Рефакторинг компонентов (разделение на более мелкие) • Финальная настройка и тестирование |

## Технологии

- **React** — библиотека для построения пользовательского интерфейса
- **React Router DOM** — маршрутизация
- **JavaScript (ES6+)** — логика приложения
- **CSS** — стилизация
- **LocalStorage** — хранение данных на клиенте

## Функционал приложения

- ✨ Создание новой заметки с заголовком и текстом
- ✏️ Редактирование существующих заметок
- 🗑️ Удаление заметок
- 💾 Автоматическое сохранение в localStorage
- 🔍 Просмотр списка всех заметок
- 📄 Детальный просмотр отдельной заметки
- 🔄 Данные сохраняются после перезагрузки страницы

## Установка и запуск

### Требования
- Node.js (версия 14 или выше)
- npm (менеджер пакетов)

### Инструкция

```bash
# 1. Клонировать репозиторий
git clone https://github.com/dreywi/4_semak_front.git

# 2. Перейти в папку проекта
cd 4_semak_front

# 3. Установить зависимости
npm install

# 4. Запустить приложение
npm start
