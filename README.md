Практическая работа №2

Сервер на Node.js + Express (CRUD для товаров)

Создан простой API для управления списком товаров. Реализованы все CRUD операции.
Файлы:

    2_prac/app.js — основной файл сервера

    2_prac/package.json — зависимости
    
    Товары (начальные данные):
    [
  { id: 101, name: 'тапочки', cost: 200 },
  { id: 102, name: 'кружка', cost: 150 },
  { id: 103, name: 'ваза', cost: 500 }
]
 Эндпоинты:
Метод	URL	Описание
GET	/items	Получить все товары
GET	/items/:id	Получить товар по ID
POST	/items	Создать новый товар
PATCH	/items/:id	Обновить товар
DELETE	/items/:id	Удалить товар

Запуск:

cd 2_prac
npm install
node app.js

Сервер доступен на http://localhost:3000

Практическая работа №5

Swagger-документация для API

К серверу из практики 4 добавлена интерактивная документация Swagger.
Файлы:

    backend/app-swagger.js — сервер с Swagger

Установленные пакеты:

    Схема Product (id, name, category, description, price, stock)

    GET /api/products — список товаров

    GET /api/products/{id} — товар по ID

    POST /api/products — создание товара

    PATCH /api/products/{id} — обновление товара

Запуск сервера с документацией:

cd backend
node app-swagger.js

    DELETE /api/products/{id} — удаление товара
npm install --save-dev swagger-jsdoc swagger-ui-express

Запуск сервера с документацией:

cd backend
node app-swagger.js

Доступ к документации:

Откройте браузер и перейдите по адресу:
http://localhost:3000/api-docs
